# 2. 機能一覧

## 2.1 分報投稿

- テキストベースで気軽に投稿できる（Twitterライクな短文投稿）
- 投稿時にプロジェクトを選択して紐づける
- 投稿時に関連タスクを紐づけることができる
- 投稿時にテンションメーター（5段階）やコメントを任意で記録できる
- 時間帯ごとのテンプレートを選択できる（朝のプランニング・夕方の振り返りなど）
- 画像・ファイルの添付に対応

## 2.2 プロジェクト管理

- プロジェクトを作成・編集・削除できる
- プロジェクトごとに分報を管理する
- 公開範囲を以下から選択できる
  - 非公開（自分のみ）
  - ユーザー限定公開（招待したユーザーのみ）
  - 全体公開
- 招待はメールアドレスまたはユーザーIDで行う
- プロジェクトメンバーにはロール（閲覧者 / コメント可 / 投稿可 / 管理者）を設定できる

## 2.3 タスク管理

- プロジェクト内にタスクを作成できる
- タスクのステータス管理（未着手 / 進行中 / 完了 / 保留）
- 優先度設定（高 / 中 / 低）
- 期限日の設定
- タスクに分報を紐づけることで、タスクごとの作業ログを自動集約
- かんばんボードビューとリストビューを切り替え可能

## 2.4 AI自動レポート生成

- 分報を元にAIが以下のレポートを自動生成する
  - 日報
  - 週報
  - 月報
- 生成タイミング
  - 手動トリガー
  - 定期自動生成（日次・週次・月次でスケジュール設定可能）
- レポートのフォーマットはプロンプトで自由にカスタマイズできる
- デフォルトテンプレートをプリセットとして複数用意する
  - シンプル日報（やったこと・明日の予定）
  - Scrum形式（昨日やったこと・今日やること・ブロッカー）
  - KPT形式（Keep・Problem・Try）
- 生成されたレポートは編集・保存・エクスポート（Markdown / PDF）できる

## 2.5 AIインサイト

- 分報の蓄積データを元に以下のインサイトを表示する
  - 作業時間の偏り（どのプロジェクト・タスクに時間をかけているか）
  - 進捗が滞っているプロジェクトの検出
  - テンションの推移グラフ
  - 週次・月次の活動サマリー

## 2.6 スタンドアップ自動生成

- 本日の分報から「昨日やったこと・今日やること・ブロッカー」を抽出
- Scrum形式のスタンドアップテキストを自動生成
- SlackやDiscordへのワンクリック共有に対応

## 2.7 リアクション・コメント

- 公開・限定公開プロジェクトの分報にスタンプリアクションを付けられる
- コメントを投稿できる
- メンション（@ユーザー名）に対応

## 2.8 外部連携

- Slack連携
  - 日報・週報をSlackチャンネルへ自動投稿
- Discord連携
  - 日報・週報をDiscordチャンネルへ自動投稿
- GitHub連携
  - コミット履歴を分報として自動インポート（オプション）
- カレンダー連携（Google Calendar）
  - スケジュールを分報の参考情報として表示
