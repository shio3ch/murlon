# 3. 画面構成

```
/                          ランディングページ
/login                     ログイン
/signup                    ユーザー登録
/dashboard                 ダッシュボード（全プロジェクトの最新分報）
/projects                  プロジェクト一覧
/projects/new              プロジェクト作成
/projects/:id              プロジェクト詳細（分報タイムライン）
/projects/:id/tasks        タスク管理（かんばんボード）
/projects/:id/reports      レポート一覧・生成
/projects/:id/settings     プロジェクト設定（公開範囲・メンバー管理）
/insights                  AIインサイト
/settings                  ユーザー設定
```
